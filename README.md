# HTML & Markdown WYSIWYG Editor (Editor Studio)

Современный, быстрый и отзывчивый двусторонний редактор для работы с Markdown и HTML, выполненный в стиле Material Design 3 (MD3).

## Особенности и функции

- **Двусторонняя синхронизация:**
  - Левая панель для редактирования сырого кода.
  - Правая панель для визуального (WYSIWYG) редактирования.
  - Изменения мгновенно отражаются в обеих панелях без бесконечных циклов.
- **Поддержка Markdown и HTML:**
  - Переключайтесь между режимами Markdown и HTML в левой панели одним кликом.
  - При переключении код автоматически форматируется (через `js-beautify`).
- **Синхронная прокрутка:** Пролистывание одной панели автоматически прокручивает вторую, сохраняя контекст чтения.
- **Адаптивный интерфейс (Responsive):** 
  - На широких экранах панели располагаются рядом (Side-by-side).
  - На узких экранах или разделенном мониторе панели элегантно складываются друг под друга (Stacking).
- **Темная тема (Dark Mode):** Полная поддержка темного и светлого режимов с плавными переходами цветов.
- **Безопасность:** Встроенная санитизация HTML через `DOMPurify` для защиты от XSS.
- **Экспорт в один клик:** Быстрое копирование результата в форматах Markdown или отформатированном HTML.

## Технологический стек

- **Ядро:** React 19 + Vite + TypeScript
- **Стилизация:** Tailwind CSS v4.0 (с плагином Typography)
- **Визуальный редактор:** Tiptap (ProseMirror)
- **Редактор кода:** `react-simple-code-editor` + `PrismJS` (с поддержкой подсветки синтаксиса Markdown и HTML)
- **Утилиты:** `marked`, `dompurify`, `js-beautify`, `lucide-react`

## Запуск проекта локально

1. Установите зависимости:
   ```bash
   npm install
   ```

2. Запустите сервер разработки:
   ```bash
   npm run dev
   ```

3. Откройте браузер по адресу `http://localhost:5173`.
