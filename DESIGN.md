# Проект: Editor Studio (WYSIWYG + Markdown/HTML)

Современный редактор, поддерживающий Markdown и HTML, предоставляющий двустороннюю синхронизацию с визуальным редактором.

## Стек технологий
- **Фреймворк:** React (Vite), TypeScript
- **Стилизация:** Tailwind CSS v4
- **Визуальный редактор:** Tiptap (на базе ProseMirror)
- **Редактор кода:** react-simple-code-editor + PrismJS
- **Безопасность и форматирование:** DOMPurify, marked, js-beautify
- **Иконки:** Lucide React

## Основные возможности
1. **Двустороннее редактирование:** 
   - Левая панель: Текстовый редактор для сырого Markdown ИЛИ сырого HTML (с переключателем).
   - Правая панель: Визуальный WYSIWYG редактор.
   - Изменения в одной панели мгновенно отображаются в другой.
2. **Синхронная прокрутка:** Прокрутка одной панели автоматически перемещает вторую для сохранения контекста.
3. **Общая панель инструментов:** Форматирование применяется к визуальному редактору и автоматически отражается в коде.
4. **Темы оформления:** Поддержка светлой и темной тем в стиле Material Design 3 (MD3).
5. **Адаптивность:** Панели располагаются горизонтально на ПК и вертикально на узких экранах.

## Пользовательский интерфейс (UI)
- **Шапка (Floating App Bar):** Содержит заголовок, переключатель тем и кнопки экспорта.
- **Панель инструментов:** Закругленная панель с кнопками форматирования.
- **Основная область:** Две карточки со скруглениями 32px (в стиле MD3).
- **Статус-бар:** Отображает статистику символов и текущий режим работы.
